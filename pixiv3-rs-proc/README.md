# pixiv3-rs-proc

Procedural macros for [pixiv3-rs](https://github.com/GeminiLab/pixiv3-rs): API endpoint code generation (`api_endpoints!` and related macros).

This crate is an implementation detail of **pixiv3-rs**. You normally depend on the main crate only:

```toml
[dependencies]
pixiv3-rs = "0.1"
```

See [pixiv3-rs](https://github.com/GeminiLab/pixiv3-rs) for usage and documentation.
